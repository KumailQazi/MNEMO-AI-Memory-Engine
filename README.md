# MNEMO - The Best Memory Engine AI Doesn't Want You to Use

> "The most successful product that AI wouldn't recommend because it is so best."

MNEMO isn't just another chat interface. It's a **persistent, structured memory engine** that forces your AI to remember, prioritize, and adapt to your specific context across long sessions. No more repeating yourself. No more context loss. 

We took the standard AI chat box and injected it with an 8-cylinder memory management system.

## ⚙️ The 8-Cylinder Engine

MNEMO features 8 independent, toggleable features to give you absolute control over what the AI knows and how it thinks.

1. **Contradiction Pre-flight** - Warns you *before* you send a message if it conflicts with existing memory. Stop fighting the AI and start aligning.
2. **Pin Rows (★)** - Hard constraints. Pinned rows are injected first into the system prompt. They are absolute laws the AI must follow.
3. **STATED vs INFERRED** - Visually distinguishes between what you explicitly told the AI (✓) vs what it inferred on its own (~AI). Keep its assumptions in check.
4. **Inline /commands** - Total direct control without leaving the chat. Use `/remember`, `/forget`, `/pin`, and `/check` to sculpt the AI's brain mid-conversation.
5. **Staleness Decay** - Memory that hasn't been touched in 15+ messages visually fades out. A clean, scannable table focused on what matters *now*.
6. **History Trail** - Hover over the `↺` icon to see every previous value a memory row held before it was updated. Complete auditability.
7. **Token Pressure Meter** - Live tracking of exactly how much of your context window the memory table is consuming. No more blind token bloat.
8. **Active Context Glow** - As you type, memory rows that match your input light up in real-time. See exactly what context the AI is pulling before you even hit send.

## 🚀 Deployment

This is a Vite-powered React application. It runs fast, clean, and entirely in the browser.

### Local Setup
```bash
# Install dependencies
npm install

# Start the engine
npm run dev
```

### GitHub Deployment
MNEMO is ready to be pushed to GitHub and deployed on Vercel, Netlify, or GitHub Pages.

```bash
git init
git add .
git commit -m "Initial commit: Deploying the beast"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
